# Test technique pour un stage chez Uballers

**Pour installer le repository en local, vous pouvez faire un git clone.**
Il vous faudra ensuite utiliser la commande npm install, pour récupérer tous les modules nécessaires.
Enfin, avec la commande npm start, vous pourrez visualiser l'application en localhost.

**Mais vous pouvez également consulter l'application directement en ligne.**
Il vous suffit de suivre le lien suivant : https://uballers-test.netlify.app/
